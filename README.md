<h1 align="center">BlackTech</h1>

### 📚 **SOBRE**

Este projeto é um sistema de e-commerce desenvolvido em Java para aprimorar habilidades de programação web. O objetivo principal é oferecer uma plataforma onde os usuários possam explorar uma variedade de Produtos, adiciona-los ao carrinho e finalizar suas compras de forma prática e segura.

#
### 🎯 **OBJETIVO**

O desenvolvimento deste sistema tem como finalidade aplicar conceitos de programação web full stack, incluindo autenticação de usuários, segurança de dados, integração entre front-end e back-end, além do gerenciamento de informações em banco de dados relacional.

#
### 🔧 **FUNCIONALIDADES**

- **Cadastro e Login de Usuários:** Permite aos clientes criar uma conta e acessar suas funcionalidades personalizadas.

- **Listagem de Produtos:** Exibe produtos disponíveis, com informações detalhadas como marca, preço e especificações.

- **Carrinho de Compras:** Possibilita adicionar e remover produtos, atualizar quantidades e visualizar o valor total.

- **Checkout:** Simula o processo de finalização do pedido, gerando um resumo de compra.

- **Administração:** Permite que administradores gerenciem produtos (cadastrar, editar e excluir).
